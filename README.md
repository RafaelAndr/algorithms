# Projeto e Análise de Algoritmos

Repositório contendo os códigos desenvolvidos na disciplina **Projeto e Análise de Algoritmos**, ministrada na **Universidade Federal de Sergipe (UFS)**.

**Docente:** Bruno Otávio de Piedade Prado

## 📚 Algoritmos implementados

Os seguintes algoritmos foram utilizados no desenvolvimento dos projetos:

1. **Merge Sort**
2. **Quick Sort**
3. **Heap Sort**
4. **Programação Dinâmica**
5. **Busca em Cadeias**
6. **Compressão**

## 🎯 Objetivo

Este repositório tem como objetivo reunir implementações e experimentos práticos relacionados aos principais algoritmos estudados na disciplina, explorando conceitos como:

* Complexidade de algoritmos
* Paradigmas de projeto de algoritmos
* Eficiência computacional
* Estruturas de dados aplicadas
